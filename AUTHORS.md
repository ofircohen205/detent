## How to Get Attribution

### For Contributors

If you've contributed code, documentation, or other work to Detent, you can request attribution!

Submit a pull request that adds your name and role to the appropriate section below:

**Code Contributors:**

```markdown
- **Your Name** — Brief description of contribution (PR #123)
```

**Documentation Contributors:**

```markdown
- **Your Name** — Documentation or guides
```

**Translators:**

```markdown
- **Your Name** — [Language] translation
```

### For Security Researchers

Found a vulnerability? We'll credit you in SECURITY.md and CHANGELOG.md upon patch release (if you'd like).

### For Design & UX

Contributing design feedback, wireframes, or UX improvements? Let's discuss in GitHub Discussions.
